# Admin Tools Frontend

Frontend base en Next.js + React + TypeScript.

## Requisitos

- Node.js 18+
- npm o yarn

## Instalacion

```bash
npm install
```

## Ejecucion local

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Build para produccion

```bash
npm run build
npm start
```
